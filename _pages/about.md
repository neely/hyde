This is a [Hyde](https://github.com/YOUR-GITHUB-USERNAME/YOUR-GITHUB-USERNAME.github.io) blog -- a fork of the [Reverie](https://github.com/amitmerchant1990/reverie) Jekyll theme with an added mobile post composer.

Replace this page with a real "about" for your own blog once you're set up.

## Features

- Fully responsive, mobile-optimized theme
- Free hosting on GitHub Pages
- Markdown blogging with Kramdown / GFM
- Syntax highlighting (Dracula theme included)
- Fuzzy search across posts
- Pagination, categories, RSS, and sitemap out of the box
- A standalone post composer (/compose.html) for writing and publishing from any device, with drafts, image paste, and GIF support
